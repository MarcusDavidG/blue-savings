# BlueSavings Frontend

Next.js 14 frontend for BlueSavings decentralized savings protocol.

## Quick Start

```bash
npm install
npm run dev
```

## Features

- Wallet connectivity with RainbowKit
- Contract interaction with wagmi
- Real-time vault updates
- Mobile-responsive design
- Dark mode support

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- wagmi & viem
- RainbowKit
- React Query
