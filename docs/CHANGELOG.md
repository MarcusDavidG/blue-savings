# Changelog

All notable changes to BlueSavings will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-12

### Added
- Comprehensive documentation structure
- Helper scripts for mainnet interaction
- Integration test suite
- Fuzz testing infrastructure
- Invariant tests
- CI/CD workflows for automated testing
- Gas optimization improvements
- 19 view/helper functions

### Changed
- Enhanced NatSpec documentation
- Improved error messages
- Updated README with examples

### Fixed
- Minor compiler warnings
- Test edge cases

## [1.0.0] - 2026-01-09

### Added
- Initial SavingsVault contract
- Time-locked vaults
- Goal-based vaults
- Vault metadata
- Protocol fees (0.5%)
- Emergency withdrawals
- Comprehensive test suite (58 tests)
- Deployment to Base mainnet and Sepolia
- BaseScan verification
